API Overview

RESTful API

Built with Django REST Framework for robust and scalable endpoints

JWT Auth

Secure authentication using JSON Web Tokens

High Performance

Optimized queries and caching for fast responses

The FlixReview API provides comprehensive access to our movie review platform. It supports creating, reading, updating, and deleting movies, reviews, and user data.

Base URL: http://127.0.0.1:8000/api/

Authentication

JWT Token Authentication

All authenticated endpoints require a valid JWT token in the Authorization header.

1. Register a new user

POST /api/auth/register/

2. Obtain access token

POST /api/auth/token/

3. Use token in requests

Authorization: Bearer <your_access_token>

API Endpoints

Movies

GET /api/movies/
List all movies
GET /api/movies/{id}/
Get movie details
POST /api/movies/
Create new movie

Reviews

GET /api/reviews/
List all reviews
POST /api/reviews/
Create review
PUT /api/reviews/{id}/
Update review

Recommendations

GET /api/recommendations/personalized/
Get personalized recommendations
GET /api/recommendations/similar/{movie_id}/
Get similar movies

Code Examples

Creating a Review

curl -X POST http://127.0.0.1:8000/api/reviews/ \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "movie": 1,
    "rating": 5,
    "content": "Amazing movie! Highly recommended."
  }'
